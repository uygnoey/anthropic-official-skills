## Instructions
Use this skill to extract logistics from the Code with Claude event announcement and turn them into a planning checklist.

Include:
- **Dates and cities**
- **Participation modes**: in-person (limited, application + selection) vs livestream/recordings
- **Key actions**: apply/register, watch for agenda updates, track notification timing

Do not invent details not present in the announcement.

## Examples
### Example: Attendance plan template
- Decide: in-person vs livestream
- Add the event date and local timezone
- Apply for an in-person seat (if available)
- Register for the livestream
- Set a reminder to check for the agenda preview
- Watch for attendee notification timing
