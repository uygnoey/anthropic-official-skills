## このガイドの内容
Code with Claude に現地参加する／リモート視聴するための簡単な計画です。

## 手順
- 都市／日程を選ぶ（サンフランシスコ：2026年5月6日、ロンドン：2026年5月19日、東京：2026年6月10日）。
- 現地参加の場合：席に応募する（席数は限定、抽選）。
- リモート視聴の場合：ライブ配信に登録する。
- アジェンダのプレビュー公開時期を確認する（原文：4月上旬に確認）。
- 応募結果の通知時期を確認する（原文：4月上旬）。
