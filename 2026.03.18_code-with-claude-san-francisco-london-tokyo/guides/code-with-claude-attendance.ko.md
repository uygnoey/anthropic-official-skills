## 이 가이드에서 다루는 내용
Code with Claude에 현장 참석하거나 원격으로 시청하기 위한 간단한 계획입니다.

## 단계
- 도시/일정을 선택합니다(샌프란시스코: 2026년 5월 6일, 런던: 2026년 5월 19일, 도쿄: 2026년 6월 10일).
- 현장 참석을 원하면 좌석 신청을 합니다(좌석 수가 제한되어 있으며, 무작위 추첨으로 선정).
- 원격 시청이라면 라이브스트림 등록을 합니다.
- 아젠다 프리뷰 공개 시점을 확인합니다(원문: 4월 초에 다시 확인).
- 신청자 결과 통보 시점을 확인합니다(원문: 4월 초).
