## What is this post?
This post announces Code with Claude, Anthropic’s developer conference, expanding to San Francisco, London, and Tokyo.

## When is it useful?
It’s useful if you want to learn about Claude Code via technical sessions, live demos, workshops, and office hours, either in-person (limited seats) or via livestream.

## Key points
- Cities and dates: San Francisco (May 6, 2026), London (May 19, 2026), Tokyo (June 10, 2026).
- Applications for in-person seats are open; attendees are selected by random drawing and notified in early April.
- Sessions will be livestreamed and recorded for anyone to watch.
