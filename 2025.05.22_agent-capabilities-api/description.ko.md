## 이 글이 뭔가요
이 글은 더 나은 AI 에이전트를 만들기 위한 새로운 API 기능(코드 실행 도구, MCP 커넥터, Files API, 확장 프롬프트 캐싱)을 소개합니다.

## 언제 유용한가요
Python으로 데이터를 분석하거나, MCP 서버를 통해 외부 시스템과 연결하거나, 파일을 세션 간에 저장·재사용하거나, 긴 프롬프트를 캐싱해 지연과 비용을 줄여야 할 때 유용합니다.

## 핵심 포인트
- 코드 실행은 Claude가 샌드박스 환경에서 Python을 실행해 분석과 반복 작업을 할 수 있게 합니다.
- MCP 커넥터는 별도 클라이언트 통합 코드를 작성하지 않아도 원격 MCP 서버에 연결해 도구를 사용할 수 있게 합니다.
- Files API는 문서를 한 번 업로드해 여러 세션에서 재사용할 수 있게 합니다.
- 확장 프롬프트 캐싱은 더 긴 TTL(최대 60분 포함)을 제공해 반복되는 긴 프롬프트의 비용과 지연을 줄입니다.
