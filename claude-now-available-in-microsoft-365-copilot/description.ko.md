## 이 글이 뭔가요
Microsoft가 **Microsoft 365 Copilot에서 Claude 모델을 선택해 사용할 수 있게 됐다**고 발표한 내용을 Anthropic이 요약합니다. 적용 범위는 Researcher agent와 Copilot Studio부터 시작합니다.

## 언제 유용한가요
- Microsoft 365 Copilot 환경에서 Claude를 사용하려는 엔터프라이즈 사용자/조직에게
- Researcher 또는 Copilot Studio에서 업무 목적에 맞춰 모델 선택이 필요한 경우

## 핵심 포인트
- Microsoft 365 Copilot 고객은 (롤아웃 시점 기준 Frontier Program을 통해) **Claude Sonnet 4**와 **Claude Opus 4.1**을 모델 옵션으로 선택할 수 있습니다.
- Microsoft의 **Researcher agent**는 멀티스텝 리서치를 수행하며, **Claude Opus 4.1**으로 구동될 수 있다고 설명합니다.
- **Copilot Studio**에서도 Claude 모델이 에이전트 구축/관리용 옵션으로 제공됩니다.
- Anthropic은 모델이 Microsoft 관리 환경 밖에서 호스팅되며 Anthropic 이용약관의 적용을 받는다고 언급합니다.

## 번들 리소스
- 없음(발표 성격이며, 코드/설정 템플릿은 포함되지 않습니다).
