## 개요
이 가이드는 발표 내용 기준으로 Microsoft 365 Copilot에서 Claude가 어디에 제공되는지 요약합니다.

## Claude 제공 위치
- Researcher: Microsoft의 추론 에이전트가 Claude Opus 4.1로 구동될 수 있습니다.
- Copilot Studio: 엔터프라이즈 에이전트 구축을 위해 Claude Sonnet 4와 Claude Opus 4.1이 모델 옵션으로 제공됩니다.

## 참고
- 제공은 Microsoft Frontier Program(옵트인) 기반 롤아웃으로 설명됩니다.
- Anthropic은 모델이 Microsoft 관리 환경 밖에서 호스팅되며 Anthropic 이용약관의 적용을 받는다고 언급합니다.
