## この記事について
Microsoftが **Microsoft 365 CopilotでClaudeモデルを利用可能**にしたという発表を、Anthropicが要約します。対象はResearcherとCopilot Studioから開始します。

## どんなときに役立つか
- Microsoftの生産性環境の中でClaudeにアクセスしたい企業ユーザー向け
- ResearcherやCopilot Studioで用途に応じてモデル選択が必要なとき

## 主なポイント
- Microsoft 365 Copilotの顧客は（展開時点ではMicrosoftのFrontier Programを通じて）**Claude Sonnet 4**と**Claude Opus 4.1**をモデルとして選択できます。
- Microsoftの**Researcher**は複数ステップの調査を行い、**Claude Opus 4.1**で動作し得ると説明されています。
- **Copilot Studio**でも、企業向けエージェントの作成/管理のためにClaudeモデルが選択肢として提供されます。
- Anthropicは、モデルがMicrosoft管理環境の外でホスティングされ、Anthropicの利用規約の適用対象であると述べています。
