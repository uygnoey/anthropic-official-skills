# Skills 패키징 가이드

## 개요
이 가이드는 단계적 공개를 활용해 Agent Skills를 구조화하는 글의 권장 접근을 요약합니다.

## 권장 구조
- YAML 프런트매터는 최소화(name + description).
- 핵심 실행 지침은 `SKILL.md`에.
- 장문 자료는 `references/`로 분리하고 `SKILL.md`에서 링크.

## 왜 도움이 되나요
단계적 공개는 큰 스킬 라이브러리를 사용할 수 있게 해 줍니다. 모델은 낮은 비용으로 많은 스킬 “목록”을 보고, 필요할 때만 자세한 내용을 읽습니다.
