# Claude Sonnet 4 ahora admite 1M de tokens de contexto

## ¿De qué trata este post?
Este post anuncia que Claude Sonnet 4 admite hasta 1 millón de tokens de contexto en la API de Anthropic, y resume lo que eso habilita.

## ¿Cuándo es útil?
Es útil cuando quieres analizar entradas muy grandes en una sola solicitud (por ejemplo, bases de código completas o grandes conjuntos de documentos) o construir agentes que mantengan coherencia a lo largo de flujos de trabajo largos y de varios pasos.

## Puntos clave
- Sonnet 4 admite hasta 1M de tokens de contexto (un aumento de 5×).
- Casos de uso: análisis de código a gran escala, síntesis de documentos y agentes conscientes del contexto.
- El contexto largo está en beta pública en la Claude Developer Platform y también está disponible en Amazon Bedrock y Google Cloud Vertex AI.
- Cambios de precio aplican a prompts de más de 200K tokens; el caching de prompts y el procesamiento por lotes pueden reducir latencia/costo.
