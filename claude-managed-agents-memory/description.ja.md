## この記事について
この記事は、Claude Managed Agents におけるファイルシステムベースの組み込みメモリがパブリックベータで利用可能になったことを告知します。

## どんなときに役立つか
独自のメモリ基盤を構築・運用せずに、セッションをまたいで改善する本番向けエージェントを作りたいときに役立ちます。

## 主なポイント
- メモリは、セッションをまたいで学習し、複数エージェント間で学びを共有できる長時間稼働エージェントを想定して設計されています。
- メモリはファイルシステムに直接マウントされるため、エージェントは慣れたツール（bash とコード実行）でメモリを扱えます。
- メモリは持ち運び可能なファイルとして保存され、エンタープライズ向けの制御（権限スコープ、監査ログ、エクスポート／ロールバック／レダクション、API によるプログラム管理）を提供します。
- 更新は Claude Console 上でセッションイベントとしても表示され、追跡可能性が高まります。
