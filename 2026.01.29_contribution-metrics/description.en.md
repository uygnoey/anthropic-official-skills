## What is this post?
This post announces contribution metrics (public beta) in Claude Code, which connect to GitHub to help teams measure Claude Code’s impact on shipping velocity.

## When is it useful?
Useful when you want quantitative adoption/impact signals (e.g., PR throughput and commit activity) for Claude Code across an engineering organization.

## Key points
- Contribution metrics integrate with GitHub to surface pull requests merged, code committed, and per-user contribution data.
- Metrics are derived by matching Claude Code session activity with GitHub PR/commit activity, using conservative, high-confidence attribution.
- The metrics appear in the Claude Code analytics dashboard for workspace admins/owners.
