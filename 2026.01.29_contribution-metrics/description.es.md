## ¿De qué trata este post?
Este post anuncia las métricas de contribución (beta pública) en Claude Code, que se conectan a GitHub para ayudar a medir el impacto de Claude Code en la velocidad de entrega.

## ¿Cuándo es útil?
Es útil cuando necesitas señales cuantitativas de adopción e impacto (por ejemplo, volumen de PRs y actividad de commits) de Claude Code en una organización de ingeniería.

## Puntos clave
- Las métricas de contribución se integran con GitHub para mostrar PRs fusionadas, código comprometido y datos de contribución por usuario.
- Las métricas se calculan asociando la actividad de sesiones de Claude Code con la actividad de PR/commits en GitHub, usando una atribución conservadora y de alta confianza.
- Las métricas aparecen en el panel de analíticas de Claude Code para administradores/propietarios del espacio de trabajo.

## Recursos incluidos
- Este post no incluye plantillas, scripts ni archivos de configuración reutilizables.
