## 이 글이 뭔가요
이 글은 Claude Code의 기여도 메트릭(공개 베타)을 소개합니다. GitHub와 연결해 Claude Code가 팀의 배포/개발 속도에 미치는 영향을 측정할 수 있게 합니다.

## 언제 유용한가요
엔지니어링 조직 전반에서 Claude Code의 도입과 효과를 정량적으로 보고 싶을 때(예: PR 처리량, 커밋 활동) 유용합니다.

## 핵심 포인트
- 기여도 메트릭은 GitHub와 연동되어 병합된 PR, 커밋된 코드, 사용자별 기여 데이터를 보여줍니다.
- Claude Code 세션 활동과 GitHub의 PR/커밋 활동을 매칭해 지표를 산출하며, 보수적으로(높은 확신이 있는 경우만) 귀속합니다.
- 워크스페이스 관리자/소유자가 Claude Code 분석(analytics) 대시보드에서 확인할 수 있습니다.

## 번들 리소스
- 이 글에는 재사용 가능한 템플릿/스크립트/설정 파일이 포함되어 있지 않습니다.
