## この記事について
この記事は、Claude Code の「貢献メトリクス」（パブリックベータ）を紹介します。GitHub と連携し、Claude Code が開発・リリース速度に与える影響を測定できるようにします。

## どんなときに役立つか
エンジニアリング組織全体で、Claude Code の導入状況と効果を定量的に把握したいとき（例: PR の処理量やコミット活動）に役立ちます。

## 主なポイント
- 貢献メトリクスは GitHub と統合し、マージされた PR、コミットされたコード、ユーザー別の貢献データを可視化します。
- Claude Code のセッション活動と GitHub の PR/コミット活動を突合して算出し、保守的に（高い確度がある場合のみ）帰属します。
- ワークスペースの管理者/オーナーが Claude Code の分析ダッシュボードで確認できます。
