# Claude Sonnet 4 now supports 1M tokens of context

## What is this post?
This post announces that Claude Sonnet 4 supports up to 1 million tokens of context on the Anthropic API, and summarizes what that enables.

## When is it useful?
Useful when you want to analyze very large inputs in one request (e.g., entire codebases or large document sets) or build agents that keep coherence across long multi-step workflows.

## Key points
- Sonnet 4 supports up to 1M tokens of context (a 5× increase).
- Use cases include large-scale code analysis, document synthesis, and context-aware agents.
- Long context is available in public beta on the Claude Developer Platform and is also available via Amazon Bedrock and Google Cloud Vertex AI.
- Pricing changes apply for prompts over 200K tokens; prompt caching and batch processing can reduce latency/cost.
