## 이 글이 뭔가요
이 글은 Anthropic Growth Marketing 팀이 Claude Code로 반복적인 마케팅 실무를 줄이는 간단한 도구를 만든 사례를 소개합니다. 대표적으로 크리에이티브 변형을 한 번에 만드는 Figma 플러그인과, 반응형 검색 광고(RSA) 카피를 생성·수정한 뒤 Google Ads 업로드용 CSV로 내보내는 워크플로를 다룹니다.

## 언제 유용한가요
- 엄격한 글자수 제한 아래에서 광고 카피를 자주 만들거나 갱신해야 할 때
- 여러 종횡비(aspect ratio)의 크리에이티브 변형을 반복적으로 업데이트할 때
- 비개발자가 자연어로 문제를 설명해 작은 내부 도구를 직접 만들고 싶을 때

## 핵심 포인트
- 작은 반복 작업부터 시작해 가치 검증(작성자는 간단한 계산기 앱으로 시작).
- 문제를 자연어로 명확히 설명하고, 반복·트러블슈팅을 통해 완성도를 올립니다.
- API 문서 같은 기존 자료를 활용하고, 브랜드 톤/정확성/베스트 프랙티스를 Skill로 파일화합니다.
- 결과물은 실무에서 바로 쓰이는 산출물(예: Google Ads 업로드용 CSV)로 연결돼야 합니다.
