## この記事について
この記事では、Anthropic の Growth Marketing チームが Claude Code を使って、反復的なマーケティング作業を減らす小さなツールを構築した事例を紹介します。具体的には、クリエイティブのバリエーションを一括生成する Figma プラグインと、レスポンシブ検索広告（RSA）のコピーを作成・改善し、Google Ads にアップロード可能な CSV に書き出すワークフローが中心です。

## どんなときに役立つか
- 厳しい文字数制限のもとで広告コピーを頻繁に作成・更新するとき
- 複数のアスペクト比に対してクリエイティブのバリエーション更新を繰り返すとき
- 非エンジニアが自然言語で課題を説明し、小さな社内ツールを自作したいとき

## 主なポイント
- 小さく反復的なステップから始めて価値を検証する（筆者はシンプルな電卓アプリから開始）。
- 課題を自然言語で明確に伝え、反復とトラブルシュートで仕上げる。
- 既存リソース（API ドキュメント等）を活用し、ブランドトーン、製品の正確性、ベストプラクティスを Skills としてファイル化する。
- 出力は実務で使える成果物（例：Google Ads 用のアップロード可能な CSV）につなげる。
