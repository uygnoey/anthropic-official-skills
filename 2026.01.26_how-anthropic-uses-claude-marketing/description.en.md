## What is this post?
This post describes how Anthropic’s Growth Marketing team uses Claude Code to build lightweight tools that remove repetitive marketing busywork—specifically a Figma plugin for creative variations and a responsive search ads (RSA) copy workflow that exports upload-ready CSV.

## When is it useful?
- You frequently create or refresh ad copy under strict character limits.
- You repeatedly update creative variations across multiple aspect ratios.
- You want non-engineers to build small internal tools by describing problems in natural language.

## Key points
- Start with small, repetitive steps to validate value (the author began by building a simple calculator app).
- Describe your problem clearly in natural language; iterate and troubleshoot.
- Reuse existing resources (e.g., API docs) and encode brand/accuracy/best-practice guidance as Skills.
- Outputs should be practical deliverables (e.g., CSV ready for Google Ads upload).
