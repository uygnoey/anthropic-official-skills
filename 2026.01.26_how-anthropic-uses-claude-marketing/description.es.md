## ¿De qué trata este post?
Este post cuenta cómo el equipo de Growth Marketing de Anthropic usa Claude Code para construir herramientas pequeñas que eliminan trabajo repetitivo de marketing—en particular, un plugin de Figma para generar variaciones creativas y un flujo para anuncios de búsqueda responsivos (RSA) que termina exportando un CSV listo para subir a Google Ads.

## ¿Cuándo es útil?
- Si creas o actualizas copy de anuncios con frecuencia bajo límites estrictos de caracteres.
- Si actualizas variaciones creativas a través de múltiples relaciones de aspecto de forma repetitiva.
- Si quieres que personas no técnicas creen herramientas internas describiendo el problema en lenguaje natural.

## Puntos clave
- Empieza con pasos pequeños y repetitivos para validar el valor (el autor comenzó con una app de calculadora simple).
- Describe el problema claramente en lenguaje natural; itera y depura.
- Reutiliza recursos existentes (p. ej., documentación de APIs) y captura tono de marca, exactitud de producto y buenas prácticas como Skills.
- Los resultados deben ser entregables prácticos (p. ej., CSV listo para cargar en Google Ads).
