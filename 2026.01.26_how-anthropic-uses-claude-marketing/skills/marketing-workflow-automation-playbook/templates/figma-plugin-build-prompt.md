# Figma plugin build prompt (template)

## Prompt (from the post)
Claude, I'm working in Figma. I really want to be able to solve this challenge of this repetitive copy and pasting. Can you help me build a Figma plugin to help me solve my challenge?

## Additions to make it actionable
- Describe the frames/aspect ratios you need to support.
- Define how variations are provided (e.g., multi-line text box, CSV paste).
- Define the “permutations” rules (e.g., which text maps to which layers).
- Ask for installation steps for the plugin.
