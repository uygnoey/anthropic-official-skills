# RSA copy workflow spec (template)

This template captures the “/rsa” workflow described in the post.

## Invocation
- Slash command: `/rsa`

## Inputs
- Campaign data (high-performing themes, audience info, etc.)
- Existing copy (headlines/descriptions)
- Keywords

## Guardrails (Agent Skills)
- Brand tone and voice
- Product accuracy
- Google Ads RSA best practices

## Process
1) Brainstorm candidate headlines and descriptions.
2) Validate strict character counts.
3) Iterate with the user to refine (treat output as a starting point).
4) Export a CSV that is ready for Google Ads upload.
