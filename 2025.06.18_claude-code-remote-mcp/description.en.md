## What is this post?
This post announces remote MCP server support in Claude Code, enabling secure connections to external tools and data sources without running local servers.

## When is it useful?
Use remote MCP servers when you want Claude Code to access third-party tools (and their resources) while keeping setup low-maintenance and avoiding local server management.

## Key points
- Claude Code can connect to remote MCP servers to access both tools and resources exposed by the server.
- Remote MCP servers reduce maintenance because vendors handle updates, scaling, and availability.
- Claude Code supports native OAuth for remote MCP servers, reducing the need for API keys or stored credentials.
