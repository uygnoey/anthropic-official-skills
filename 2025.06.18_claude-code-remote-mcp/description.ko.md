## 이 글이 뭔가요
이 글은 Claude Code에서 원격 MCP 서버를 지원한다는 소식을 소개합니다. 로컬 서버를 직접 운영하지 않고도 외부 도구와 데이터 소스를 안전하게 연결할 수 있습니다.

## 언제 유용한가요
Claude Code가 서드파티 서비스의 도구/리소스 컨텍스트를 사용하도록 하되, 로컬 서버 관리 부담을 줄이고 유지보수 비용을 낮추고 싶을 때 유용합니다.

## 핵심 포인트
- Claude Code는 원격 MCP 서버가 노출하는 도구와 리소스 모두에 접근할 수 있습니다.
- 원격 MCP 서버는 벤더가 업데이트/스케일링/가용성을 관리하므로 유지보수 부담이 줄어듭니다.
- Claude Code는 원격 MCP 서버용 네이티브 OAuth를 지원해 API 키나 자격증명 저장 필요를 줄입니다.
