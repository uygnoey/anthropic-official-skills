## ¿De qué trata este post?
Este post anuncia actualizaciones de Cowork para equipos de finanzas, incluyendo nuevos plugins orientados a finanzas y nuevos conectores MCP (FactSet y MSCI), además de flujos de trabajo que abarcan Excel y PowerPoint.

## ¿Cuándo es útil?
Es útil cuando el trabajo financiero atraviesa varias herramientas (fuentes de datos, modelos en Excel, presentaciones en PowerPoint) y quieres que Claude mantenga el contexto entre pasos usando plugins específicos del dominio.

## Puntos clave
- Claude puede trabajar entre Excel y PowerPoint manteniendo el contexto en tareas de varios pasos.
- Se presentan cinco nuevos plugins de finanzas (análisis financiero, banca de inversión, research de renta variable, private equity, gestión patrimonial).
- Se añaden conectores MCP nuevos: FactSet y MSCI, y se mencionan plugins de socios (LSEG y S&P Global).

## Recursos incluidos
- El artículo no incluye muestras de código ni manifiestos de plugins.
