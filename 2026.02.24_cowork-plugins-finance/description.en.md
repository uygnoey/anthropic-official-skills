## What is this post?
This post announces Cowork updates for finance teams, including new finance-focused plugins and new MCP connectors (FactSet and MSCI), plus workflows that span Excel and PowerPoint.

## When is it useful?
It’s useful when finance work spans multiple tools (data sources, Excel models, PowerPoint decks) and you want Claude to carry context across those steps while using domain-specific plugins.

## Key points
- Claude can work across Excel and PowerPoint while keeping context for multi-step tasks.
- Five new finance plugins are introduced (financial analysis, investment banking, equity research, private equity, wealth management).
- New MCP connectors include FactSet and MSCI, and partner plugins are mentioned (LSEG and S&P Global).
