# Verification checklist

Use this checklist during the **Verify** phase of the agent loop.

## Layout (for UI / HTML / documents)
- Are elements positioned correctly?
- Is spacing appropriate and consistent?

## Styling
- Are colors, fonts, and formatting as intended?
- Are headings and emphasis applied correctly?

## Content hierarchy
- Is information in the right order with proper emphasis?
- Are critical warnings or calls-to-action prominent?

## Responsiveness
- Does the layout break or become cramped at smaller widths?
- Are long strings (URLs, code) wrapped safely?
