# Patrones de flujo de trabajo para agentes (secuencial, paralelo, evaluador–optimizador)

## Cuándo usar cada patrón
- **Secuencial**: cuando los pasos dependen de salidas anteriores.
- **Paralelo**: cuando las subtareas son independientes y la latencia importa.
- **Evaluador–optimizador**: cuando la calidad del primer intento no basta y puedes definir criterios de evaluación.
