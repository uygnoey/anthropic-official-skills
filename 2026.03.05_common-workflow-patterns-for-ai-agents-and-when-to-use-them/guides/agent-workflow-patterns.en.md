# Agent workflow patterns (sequential, parallel, evaluator–optimizer)

## When to use each pattern
- **Sequential**: when steps depend on prior outputs.
- **Parallel**: when subtasks are independent and latency matters.
- **Evaluator–optimizer**: when first-pass quality is not sufficient and you can define evaluation criteria.
