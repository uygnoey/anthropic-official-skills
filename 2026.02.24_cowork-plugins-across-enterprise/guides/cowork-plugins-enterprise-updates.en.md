## What is this post?
This post describes Cowork and plugin updates for enterprises, focusing on admin controls (marketplaces and connector management) and expanded availability of plugins/connectors across departments.

## When is it useful?
Use these concepts when you need to operationalize Claude across many teams with centralized governance, while still allowing role-specific customization.

## Key points
- Private, organization-specific plugin marketplaces help standardize distribution.
- Plugins are portable, file-based bundles that can package skills and connectors.
- Cross-app workflows (for example, Excel to PowerPoint) can carry context between tools.
