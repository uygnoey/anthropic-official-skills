## 이 글이 뭔가요
이 글은 “Built with Opus 4.6” Claude Code 해커톤의 수상작을 발표하고, 각 프로젝트가 무엇을 만들었는지 요약합니다.

## 언제 유용한가요
- Claude Code로 무엇을 만들 수 있는지(비전문 개발자 포함) 사례를 보고 싶을 때
- 에이전트 기반 아키텍처(예: 문서 분석을 위한 병렬 sub-agent 활용)에서 영감을 얻고 싶을 때

## 핵심 포인트
- 5개의 수상 프로젝트(CrossBeam, Elisa, PostVisit.ai, TARA, Conductr)를 소개합니다.
- 해커톤 총 상금은 Claude API 크레딧 $100,000입니다.
- 일부 수상작은 에이전트 기반 접근을 언급합니다(예: 문서 파싱을 위한 병렬 sub-agent, 스펙을 작업 그래프로 분해하는 meta-planner).

## 번들 리소스
- Skill: “프로젝트 사례”를 재사용 가능한 Claude Code 구현 노트로 정리하는 구조
- References: 글에서 추출한 프로젝트 요약 표
