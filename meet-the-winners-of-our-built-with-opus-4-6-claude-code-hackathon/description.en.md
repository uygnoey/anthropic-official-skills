## What is this post?
This post announces the winners of the “Built with Opus 4.6” Claude Code hackathon and summarizes what each winning project built.

## When is it useful?
- When you want examples of what people built with Claude Code (including non-professional developers).
- When you want inspiration for agent-based architectures (e.g., using parallel sub-agents for document analysis).

## Key points
- Five winning projects are highlighted (CrossBeam, Elisa, PostVisit.ai, TARA, Conductr).
- The hackathon awarded a total of $100,000 in Claude API credits.
- Some winners describe agent-based approaches (e.g., parallel sub-agents for parsing documents; a meta-planner that decomposes a spec into a task graph).
