## 이 글이 뭔가요
    이 글은 Piloting Claude in Chrome 업데이트를 소개하고, 새 기능과 제공 범위를 요약합니다.

    ## 언제 유용한가요
    브라우저 확장 프로그램에서 Claude를 안전하게 사용하기 위한 원칙(프롬프트 인젝션, 민감 행동 방지 등)을 검토할 때 유용합니다.

    ## 핵심 포인트
    - The Chrome extension lets users ask Claude to take actions in the browser, with site-level permissions.
- High-risk actions require confirmation; an experimental autonomous mode adds additional safeguards.
- Blocked high-risk site categories and classifiers help reduce prompt injection and suspicious behaviors.
- The post reports reductions in attack success rates with mitigations (autonomous mode and browser-specific attacks).
- Teams/Enterprise admins can manage org-wide settings like allowlists/blocklists.

    ## 번들 리소스
    - User guidance: start with trusted sites and avoid sensitive domains (financial/legal/medical).
- Security concepts: permissions, confirmations, classifiers, blocked categories, and mitigations against browser-specific attacks.
- Integration mention: Claude Code can pair terminal work with browser verification and debugging.
