## Qué cubre esta guía
Una lista de verificación breve para adoptar memoria basada en sistema de archivos en Claude Managed Agents.

## Lista de verificación
- Define qué debe recordar el agente entre sesiones y qué no.
- Elige la estructura de stores y sus ámbitos (por ejemplo, uno de toda la organización solo lectura vs uno por usuario con lectura/escritura).
- Planifica controles operativos: exportación, auditoría, rollback y redacción.
- Verifica la trazabilidad en la Claude Console mediante eventos de sesión.
