## What this guide covers
A short checklist for adopting filesystem-based memory in Claude Managed Agents.

## Checklist
- Define what the agent should remember across sessions vs what it should not.
- Choose store structure and scopes (e.g., org-wide read-only vs per-user read/write).
- Plan operational controls: export, auditability, rollback, and redaction.
- Verify traceability in the Claude Console via session events.
