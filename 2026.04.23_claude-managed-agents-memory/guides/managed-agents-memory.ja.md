## このガイドの内容
Claude Managed Agents のファイルシステムベースのメモリを導入するための短いチェックリストです。

## チェックリスト
- セッションをまたいで記憶すべき内容／記憶しない内容を定義する。
- ストア構成と権限スコープを決める（例：組織共通の読み取り専用 vs ユーザー別の読み取り／書き込み）。
- 運用上の制御を計画する：エクスポート、監査、ロールバック、レダクション。
- Claude Console でセッションイベントとして追跡できることを確認する。
