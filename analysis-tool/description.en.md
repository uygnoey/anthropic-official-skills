## What is this post?
This post announces that Claude.ai now includes an “analysis tool” that can run JavaScript code inside Claude.ai.

## When is it useful?
Use it when you need mathematically precise calculations or to analyze data and get exact results.

## Key points
- Claude can write and run JavaScript in Claude.ai via the analysis tool.
- The analysis tool supports precise data analysis and real-time insights.
- You can enable it from Claude.ai feature previews.
