## ¿De qué trata este post?
Este post anuncia que Claude.ai ahora incluye una “analysis tool” que puede ejecutar código JavaScript dentro de Claude.ai.

## ¿Cuándo es útil?
Es útil cuando necesitas cálculos matemáticamente precisos o analizar datos para obtener resultados exactos.

## Puntos clave
- Claude puede escribir y ejecutar JavaScript en Claude.ai mediante la analysis tool.
- La analysis tool está orientada a análisis de datos precisos y a generar insights en tiempo real.
- Se puede habilitar desde las funciones en vista previa (feature previews) en Claude.ai.
