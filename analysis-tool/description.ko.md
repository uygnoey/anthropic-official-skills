## 이 글이 뭔가요
이 글은 Claude.ai에 “analysis tool”이 추가되어, Claude가 Claude.ai 안에서 JavaScript 코드를 작성하고 실행할 수 있게 되었음을 소개합니다.

## 언제 유용한가요
수학적으로 정확한 계산이 필요하거나, 데이터를 분석해 정확한 결과를 얻고 싶을 때 유용합니다.

## 핵심 포인트
- analysis tool을 통해 Claude.ai에서 JavaScript 코드를 작성·실행할 수 있습니다.
- 정밀한 데이터 분석과 실시간 인사이트 제공을 목표로 합니다.
- Claude.ai의 기능 미리보기(Feature previews)에서 활성화할 수 있습니다.
