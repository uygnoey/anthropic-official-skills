# Build and share AI-powered apps with Claude

## この記事について
本記事は、Claude アプリ内で Claude API を呼び出すインタラクティブなアプリ（Claude-powered artifacts）を作成・ホスト・共有できる機能を紹介し、共有時は各ユーザーが自分の Claude アカウントで認証し、利用量は各自のサブスクリプションに計上され、API キー管理が不要であることを説明します。

## どんなときに役立つか
インフラを用意せずに Claude 内で素早く AI 搭載の体験をプロトタイプして共有したいとき、または共有・課金の仕組みをチームに説明したいときに役立ちます。

## 主なポイント
- Artifacts は Claude 内で Claude API を呼び出すインタラクティブなアプリになり得ます。
- 共有されたアプリは閲覧者が自分の Claude アカウントで認証し、利用量は閲覧者のサブスクリプションに計上され、作成者は費用負担せず、API キーも不要です。
- Claude が実際のコードを生成・デバッグ支援し、誰でも artifact をフォークしてカスタマイズできます。
- 当時の制約として、外部 API 呼び出し不可、永続ストレージなし、テキスト completion API に限定などが挙げられています。

## 同梱リソース
- Skill: Claude-powered artifacts の仕組み（共有／認証／課金）と制約を説明する再利用可能なプロダクトブリーフ。
