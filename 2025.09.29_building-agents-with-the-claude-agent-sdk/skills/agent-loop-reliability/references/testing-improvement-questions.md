# Questions to improve an agent (checklist)

Use these prompts when an agent’s performance is inconsistent.

- If the agent misunderstands the task: can you change the search approach so the right info is easier to find?
- If the agent fails repeatedly: can you add a formal rule or validator that identifies and fixes the issue?
- If the agent can’t fix errors: can you provide more useful tools, or tools that enable more creative exploration?
- If performance varies: can you build a representative test set/eval based on real customer usage?
