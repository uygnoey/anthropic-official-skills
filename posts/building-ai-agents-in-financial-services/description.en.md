# What is this skill?
This skill helps you plan and deploy AI agents in financial services by focusing on system integration, regulatory requirements, and controls for real-time risk.

# When should I use it?
- When you’re moving from a pilot to production for an agent in banking, insurance, or wealth management.
- When the workflow touches transactions, compliance, fraud, or other high-stakes decisions.

# Core principles
- Start with high-impact, low-risk use cases and expand gradually.
- Integrate cleanly with legacy systems and fragmented data.
- Build observability and audit trails from day one.
- Use transparent reasoning, escalation, overrides, and fail-safe defaults.

# Examples
- A fraud monitoring workflow that flags suspicious patterns and routes recommendations to analysts.
- A compliance workflow that tracks deadlines and escalates exceptions.

# Checklist / tips
- Define what actions require human approval.
- Ensure customer transparency and an easy path to a human for customer-facing agents.
