# ¿De qué trata este post?
This post announces a redesigned Claude Code desktop app aimed at running more tasks in parallel, with a new session sidebar, drag-and-drop multi-pane layout, integrated terminal and file editor, and performance improvements.

# ¿Cuándo es útil?
It is useful when you are working across multiple repos or long-running tasks and need a fast way to switch sessions, branch discussions, run commands, and review outputs without leaving the desktop app.

# Puntos clave
- Manage multiple active/recent sessions from a sidebar and filter/group them by project.\n- Use side chat (\u2318/Ctrl + ;) to branch discussions without polluting the main thread.\n- Use the integrated terminal, file editor, diff viewer, and preview pane to iterate and review in one place.\n- Arrange panes via drag-and-drop and use keyboard shortcuts for navigation.\n- Switch between Verbose/Normal/Summary view modes.
