# Claude が会話の中でインタラクティブなビジュアルを作成します

## この記事について
この記事では、会話中に Claude が一時的なインタラクティブなグラフ・図・ビジュアライゼーションをインラインで作成し、理解を助けるチャット内機能を紹介しています。

## どんなときに役立つか
概念やデータセットをビジュアルで説明してほしいとき（例：図の作成や時間変化の可視化を依頼するとき）や、会話が続く中でビジュアルを繰り返し調整・改善したいときに役立ちます。

## 主なポイント
- これらのビジュアルはアーティファクトとは異なります。インラインで一時的なものであり、会話の展開に応じて変化したり消えたりします。
- この機能はデフォルトで有効になっており、Claude が自発的にビジュアルを作成する場合と、ユーザーが直接リクエストする場合があります。
- ビジュアルが表示された後、修正（調整・詳細な探索）をリクエストできます。
