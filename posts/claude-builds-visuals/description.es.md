# Claude crea visualizaciones interactivas directamente en tu conversación

## ¿De qué trata este post?
Este post anuncia una funcionalidad integrada en el chat que permite a Claude crear gráficos, diagramas y visualizaciones interactivos de forma temporal e inline para facilitar la comprensión durante una conversación.

## ¿Cuándo es útil?
Úsala cuando quieras que un concepto o conjunto de datos se explique de forma visual (por ejemplo, pidiendo un diagrama o una visualización que varía en el tiempo) y luego iterando sobre la visualización a medida que avanza la conversación.

## Puntos clave
- Estas visualizaciones son distintas de los artefactos: son inline y temporales, y pueden cambiar o desaparecer a medida que evoluciona la conversación.
- La función está activada por defecto; Claude puede crear una visualización de forma proactiva, o puedes pedirla directamente.
- Puedes solicitar modificaciones después de que aparezca una visualización (ajustes, exploración más profunda).
