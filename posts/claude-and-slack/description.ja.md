# この記事について
この記事は、Slack で Claude を使う 2 つの方法（Slack に Claude アプリを導入してチャンネル/スレッドで支援を受ける、Claude 側で Slack コネクターを接続して会話やファイルを検索・参照できるようにする）を説明します。

# どんなときに役立つか
チームが Slack で仕事を進めているときに、返信の下書き、議論の要約、会議準備、スレッドの意思決定をドキュメント化する作業を、ワークフローを中断せずに行いたい場合に役立ちます。

# 主なポイント
- 2 つのモード: Slack の Claude（アプリ）と Claude の Slack コネクター。
- スレッドでは、Claude がまず非公開で返信案を作成し、投稿前に確認できます。
- Claude は権限のあるチャンネル/会話のみアクセスします。
- 会議準備、プロジェクト調整、オンボーディング、ドキュメント作成などの用途が示されます。
