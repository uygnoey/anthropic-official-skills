## この記事について
このプロダクト記事は、Claude アプリの **Artifacts** を紹介します。会話から作った成果物を専用スペースで閲覧し、キュレーション例を探し、既存の artifacts をすばやくカスタマイズし、整理して共有できること、さらに（ベータとして）artifacts に AI 機能を埋め込めることを説明しています。

## どんなときに役立つか
コードを書かずに、会話だけでアイデアを共有可能なアプリ風の体験（ツール、ゲーム、学習補助など）に素早く変換したいときに役立ちます。

## 主なポイント
- Artifacts は会話をインタラクティブな成果物に変える仕組みです。
- Claude アプリに artifacts 専用スペースが追加され、キュレーション例の閲覧、既存 artifacts のカスタマイズ、ゼロからの作成、整理ができます。
- 更新情報として、モバイルアプリ対応、Team/Enterprise プランでの提供、後日の MCP と永続ストレージ対応が言及されています。

## 同梱リソース
- この記事は Claude Code 向けの再利用可能な設定（skill/hook/output style）を提供せず、Claude アプリでの機能と利用開始方法を説明します。
