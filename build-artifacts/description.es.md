## ¿De qué trata este post?
Este post de producto presenta **Artifacts** en la app de Claude: un espacio dedicado para crear, explorar, personalizar, organizar y compartir creaciones interactivas, incluyendo la posibilidad de integrar capacidades de IA dentro de los artifacts (beta).

## ¿Cuándo es útil?
Es útil cuando quieres convertir una idea en una experiencia compartible tipo app (herramientas, juegos, ayudas de estudio) mediante conversación, sin necesidad de programar.

## Puntos clave
- Artifacts convierten una conversación en una creación interactiva.
- La app de Claude incorpora un espacio de artifacts para ver ejemplos curados, personalizar artifacts existentes, crear desde cero y organizar tu trabajo.
- El post menciona actualizaciones: disponibilidad en apps móviles, en planes Team/Enterprise y, posteriormente, soporte para MCP y almacenamiento persistente.

## Recursos incluidos
- El post no incluye configuraciones reutilizables para Claude Code (skills/hooks/output styles); describe capacidades del producto y cómo acceder a ellas en la app de Claude.
