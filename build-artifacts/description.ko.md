## 이 글이 뭔가요
이 제품 소개 글은 Claude 앱의 **Artifacts**를 소개합니다. 대화로 만든 결과물을 전용 공간에서 보고, 둘러보고, 기존 결과물을 빠르게 커스터마이즈하고, 정리하고, 공유할 수 있으며(베타로) 아티팩트에 AI 기능을 직접 임베드할 수 있다고 설명합니다.

## 언제 유용한가요
코드를 직접 작성하지 않고도 대화를 통해 아이디어를 공유 가능한 앱 형태(도구, 게임, 학습 보조 등)로 빠르게 만들고 싶을 때 유용합니다.

## 핵심 포인트
- Artifacts는 대화를 인터랙티브한 결과물로 바꾸는 기능입니다.
- Claude 앱에 아티팩트 전용 공간이 추가되어, 큐레이션된 예시 탐색·기존 아티팩트 커스터마이즈·처음부터 생성·정리 기능을 제공합니다.
- 업데이트로 모바일 앱, Team/Enterprise 플랜 제공, 이후 MCP 및 영구 저장소 지원이 언급됩니다.

## 번들 리소스
- 이 글은 Claude Code용 재사용 설정(스킬/hook/output style)을 제공하지 않고, Claude 앱에서의 제품 기능과 접근 방법을 설명합니다.
