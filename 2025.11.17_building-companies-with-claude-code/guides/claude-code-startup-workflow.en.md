# Claude Code startup workflow (guide)

This guide summarizes the session-based workflow described in the post.

## Workflow
1. Research in a dedicated session; optionally use subagents for parallel investigation.
2. Plan in a separate session with phased steps.
3. Implement in its own session and verify step-by-step.

## Context hygiene
- Start a new session for major phases.
- Carry forward only distilled conclusions.
