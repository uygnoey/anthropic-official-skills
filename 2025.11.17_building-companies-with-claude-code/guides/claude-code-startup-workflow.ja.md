# Claude Codeを使うスタートアップ向けワークフローガイド

このガイドは、原文で紹介されたセッションベースの開発ワークフローを要約します。

## ワークフロー
1. 専用セッションでリサーチし、必要に応じてサブエージェントで並列調査する。
2. 別セッションでフェーズ分割した計画を作る。
3. 実装はさらに別セッションで進め、段階ごとに検証する。

## コンテキスト衛生
- 主要フェーズごとに新しいセッションを開始する。
- 次に渡すのは要点に絞った結論だけにする。
