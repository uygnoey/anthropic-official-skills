# Guía de flujo de trabajo para startups con Claude Code

Esta guía resume el flujo de trabajo basado en sesiones descrito en el post.

## Flujo
1. Investiga en una sesión dedicada; opcionalmente usa subagentes para investigar en paralelo.
2. Planifica en otra sesión con fases definidas.
3. Implementa en su propia sesión y verifica paso a paso.

## Higiene de contexto
- Inicia una nueva sesión para fases principales.
- Lleva adelante solo conclusiones destiladas.
