# Claude Code 스타트업 워크플로 가이드

이 가이드는 원문에서 제시된 세션 기반 개발 워크플로를 요약합니다.

## 워크플로
1. 전용 세션에서 리서치를 수행하고, 필요하면 서브에이전트로 병렬 조사를 합니다.
2. 별도 세션에서 단계(phase) 기반 계획을 수립합니다.
3. 구현은 또 다른 세션에서 진행하며 단계별로 검증합니다.

## 컨텍스트 위생
- 큰 단계마다 새 세션을 시작합니다.
- 다음 세션으로는 정제된 결론만 전달합니다.
