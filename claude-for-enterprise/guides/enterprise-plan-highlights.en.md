## Overview
This guide summarizes the main Claude for Enterprise capabilities mentioned in the announcement, so teams can quickly map them to internal rollout needs.

## Highlights
- Large-context collaboration: 500K context window for working with long documents and sizeable internal knowledge.
- Enterprise admin & access controls: SSO/domain capture, role-based access, and a designated primary owner.
- Compliance visibility: audit logs (announced as coming soon).
- Provisioning automation: SCIM (announced as coming soon).
- Engineering workflow integration: native GitHub integration (beta at announcement).
