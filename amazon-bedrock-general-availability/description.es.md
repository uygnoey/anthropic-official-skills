## ¿De qué trata este post?
Este post anuncia que Claude está disponible de forma general (GA) en Amazon Bedrock, lo que permite a los clientes de AWS acceder a Claude mediante un servicio totalmente administrado para crear aplicaciones de IA generativa.

## ¿Cuándo es útil?
Es útil si ya estás construyendo en AWS y quieres una manera administrada de usar los modelos de Claude, con miras a flujos de trabajo agénticos (por ejemplo, usando funciones de AWS Lambda) y personalización de nivel empresarial.

## Puntos clave
- Claude está disponible de forma general en Amazon Bedrock para clientes de AWS.
- El post menciona Agents for Amazon Bedrock (preview) que orquestan llamadas de API mediante funciones de AWS Lambda.
- El anuncio destaca opciones de personalización y ejemplos de adopción empresarial.
