## 이 글이 뭔가요
이 글은 Claude가 Amazon Bedrock에서 일반 제공(GA)되었음을 발표하며, AWS 고객이 완전관리형 서비스로 Claude를 활용해 생성형 AI 애플리케이션을 만들 수 있음을 설명합니다.

## 언제 유용한가요
AWS 위에서 이미 제품을 만들고 있고, 관리형 방식으로 Claude 모델을 사용하고 싶을 때 유용합니다. 또한 AWS Lambda 함수를 활용한 에이전트형 오케스트레이션(Agents for Amazon Bedrock 등)을 염두에 두는 경우에도 참고가 됩니다.

## 핵심 포인트
- Claude가 Amazon Bedrock에서 GA로 제공됩니다.
- 글은 AWS Lambda 함수로 API 호출을 오케스트레이션하는 Agents for Amazon Bedrock(프리뷰)을 언급합니다.
- 커스터마이징과 엔터프라이즈 도입 사례를 강조합니다.

## 번들 리소스
- Skill: 이 발표를 구현 의사결정(모델 접근, 오케스트레이션, 커스터마이징)에 연결하기 위한 간단한 체크리스트
