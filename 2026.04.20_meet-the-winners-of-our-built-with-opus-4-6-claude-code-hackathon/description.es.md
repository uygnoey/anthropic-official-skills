## ¿De qué trata este post?
Este post anuncia a los ganadores del hackathon de Claude Code “Built with Opus 4.6” y resume lo que construyó cada proyecto ganador.

## ¿Cuándo es útil?
- Cuando quieres ver ejemplos de lo que la gente construyó con Claude Code (incluyendo personas que no son desarrolladoras profesionales).
- Cuando buscas inspiración para arquitecturas basadas en agentes (p. ej., uso de subagentes en paralelo para análisis de documentos).

## Puntos clave
- Se destacan cinco proyectos ganadores (CrossBeam, Elisa, PostVisit.ai, TARA, Conductr).
- El hackathon otorgó un total de 100.000 USD en créditos de Claude API.
- Algunos ganadores describen enfoques basados en agentes (p. ej., subagentes en paralelo para parsear documentos; un meta-planificador que descompone una especificación en un grafo de tareas).

## Recursos incluidos
- Skill: Una forma estructurada de convertir “casos de estudio de proyectos” en notas reutilizables de implementación con Claude Code.
- References: Una tabla resumen de proyectos extraída del post.
