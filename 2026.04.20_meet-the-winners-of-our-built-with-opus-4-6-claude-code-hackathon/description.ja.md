## この記事について
この記事は Claude Code ハッカソン「Built with Opus 4.6」の受賞者を発表し、各受賞プロジェクトが何を作ったかを要約します。

## どんなときに役立つか
- Claude Code で何が作れるか（プロの開発者ではない参加者を含む）の事例を見たいとき
- エージェントベースのアーキテクチャ（例：文書解析における並列 sub-agent 活用）のヒントが欲しいとき

## 主なポイント
- 5つの受賞プロジェクト（CrossBeam、Elisa、PostVisit.ai、TARA、Conductr）が紹介されています。
- ハッカソンの賞金総額は Claude API クレジット 100,000 ドルです。
- 一部の受賞者はエージェントベースのアプローチに言及しています（例：文書パースのための並列 sub-agent、仕様をタスクグラフに分解する meta-planner）。
