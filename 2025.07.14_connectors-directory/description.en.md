## What is this post?
This post announces a new directory of tools/connectors that integrate with Claude, including both remote SaaS connectors (e.g., Notion, Canva, Stripe, Linear) and local desktop extensions (e.g., Figma, Socket, Prisma). 

## When is it useful?
Use this when you want to quickly discover and connect third-party tools to Claude (or install desktop extensions) to enable workflows like writing release notes from Linear, turning discussions into Notion roadmaps, generating Canva social posts from briefs, or turning Figma files into code.

## Key points
- The directory lets you browse available tools and connect them with one click (authentication for remote connectors, install for desktop extensions).
- Connectors are built by partners and cover both cloud services and local desktop integrations.

## Bundled resources
- Skill: a lightweight checklist and prompt templates for exploring a connector directory and safely scoping what data/actions a connector can access.
