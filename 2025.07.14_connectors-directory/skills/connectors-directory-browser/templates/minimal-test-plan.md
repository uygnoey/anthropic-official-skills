# Minimal connector test plan (template)

## Hypothesis
If we connect <connector>, Claude should be able to <capability> using <non-sensitive data>.

## Steps
1. Connect/install the tool.
2. Run a single, low-risk request.
3. Verify results against the acceptance criteria.

## Acceptance criteria
- Output is correct and complete.
- No unintended side effects.
- No sensitive data was accessed or exposed.

## Next expansion
List the smallest scope increase to try next (one change at a time).
