# Prompt templates

## Generate a responsive layout
Create a three-column card layout that becomes single-column on mobile.

Build a navigation that collapses to hamburger menu under 768px.

Generate a dashboard with responsive sidebar that hides on tablets.

Create a product landing page with fixed header, hero section, three feature cards, and footer. Cards should display in a row on desktop and stack on mobile.

## Ask for rationale
Why use flexbox with flex-wrap instead of CSS Grid for this navigation?

## Refactor an existing codebase
Find rigid layouts and suggest flexible alternatives.
