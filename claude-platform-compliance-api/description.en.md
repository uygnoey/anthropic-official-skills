## What is this post?
This post introduces the Claude Platform Compliance API, which gives admins programmatic access to audit logs for activity across their Claude Platform organization.

## When is it useful?
- When security or compliance teams need to review Claude API usage across an organization.
- When admins need to filter activity by time range, user, or API key.
- When an organization wants a single activity feed across multiple Claude organizations under a parent organization.

## Key points
- The Compliance API provides an activity feed for audit logging.
- Access must be enabled for your organization (contact your account team).
- Admin API keys can be used to query the activity feed endpoint.
