## この記事について
この記事は Claude Platform の Compliance API を紹介します。管理者が Claude Platform 組織全体のアクティビティに関する監査ログへ、プログラムでアクセスできるようになります。

## どんなときに役立つか
- セキュリティ/コンプライアンスチームが組織の Claude API 利用状況を監査したいとき
- 管理者が時間範囲、ユーザー、API キーでアクティビティを絞り込みたいとき
- 親組織 (parent organization) 配下の複数 Claude 組織のアクティビティを単一フィードで見たいとき

## 主なポイント
- Compliance API は監査ロギングのための activity feed を提供します。
- 利用には組織での有効化が必要です（アカウント担当に連絡）。
- admin API key で activity feed endpoint を問い合わせできます。
