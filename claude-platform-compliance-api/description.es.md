## ¿De qué trata este post?
Este post presenta la Compliance API de Claude Platform, que permite a los administradores acceder de forma programática a los registros de auditoría de la actividad en toda su organización de Claude Platform.

## ¿Cuándo es útil?
- Cuando los equipos de seguridad o cumplimiento necesitan revisar el uso de Claude API en una organización.
- Cuando los administradores necesitan filtrar la actividad por rango de tiempo, usuario o clave de API.
- Cuando una organización quiere un único feed de actividad a través de varias organizaciones de Claude bajo una organización principal (parent organization).

## Puntos clave
- La Compliance API proporciona un feed de actividad para auditoría.
- El acceso debe habilitarse para tu organización (contacta a tu equipo de cuenta).
- Las claves de API de administrador pueden usarse para consultar el endpoint del feed de actividad.

## Recursos incluidos
- Skill: Un flujo reutilizable para habilitar y usar la Compliance API.
