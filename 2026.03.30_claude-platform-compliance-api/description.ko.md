## 이 글이 뭔가요
이 글은 Claude Platform Compliance API를 소개합니다. 이 API는 관리자가 Claude Platform 조직 전반의 활동에 대한 감사(audit) 로그에 프로그래밍 방식으로 접근할 수 있게 해줍니다.

## 언제 유용한가요
- 보안/컴플라이언스 팀이 조직의 Claude API 사용을 검토해야 할 때
- 관리자가 시간 범위, 사용자, API 키로 활동을 필터링해야 할 때
- 상위 조직(parent organization) 아래 여러 Claude 조직의 활동을 단일 피드로 보고 싶을 때

## 핵심 포인트
- Compliance API는 감사 로깅을 위한 activity feed를 제공합니다.
- 조직에서 사용하려면 먼저 활성화가 필요합니다(계정 담당자에게 문의).
- admin API key로 activity feed endpoint를 조회할 수 있습니다.
