# このスキルについて
このスキルは、金融サービス（銀行・保険・資産管理）における AI エージェントの計画・導入を支援します。システム統合、規制要件、リアルタイムリスク管理のための制御に重点を置いています。

# どんなときに役立つか
- 銀行・保険・資産管理分野のエージェントをパイロットから本番環境に移行するとき。
- ワークフローに取引・コンプライアンス・不正検知など高リスクな意思決定が含まれるとき。

# 基本原則
- 高インパクト・低リスクのユースケースから始め、段階的に拡張する。
- レガシーシステムや分散したデータとクリーンに統合する。
- 初日からオブザーバビリティと監査証跡を組み込む。
- 透明性のある推論・人へのエスカレーション・オーバーライド・フェールセーフのデフォルト値を活用する。

# 主なポイント（例）
- 不審なパターンを検出してアナリストへ推奨事項をルーティングする、不正監視ワークフロー。
- 期限を追跡し例外をエスカレートする、コンプライアンス管理ワークフロー。

# チェックリスト / ヒント
- どのアクションに人間の承認が必要かを定義する。
- 顧客向けエージェントでは、AI 利用の透明性を確保し、人間のオペレーターへの簡単な移行経路を用意する。
