# Build and share AI-powered apps with Claude

## 이 글이 뭔가요
이 글은 Claude 앱 안에서 Claude API를 호출하는 ‘Claude-powered artifacts’(대화형 앱)를 만들고 호스팅·공유할 수 있는 기능을 소개하며, 공유 시 각 사용자가 자신의 Claude 계정으로 인증하고 사용량은 각자의 구독에 반영되어 API 키 관리가 필요 없음을 설명합니다.

## 언제 유용한가요
인프라를 별도로 구축하지 않고 Claude 안에서 빠르게 AI 기반의 대화형 경험을 프로토타이핑하고 공유하려 할 때, 또는 공유/과금 메커니즘을 팀에 설명해야 할 때 유용합니다.

## 핵심 포인트
- Artifacts가 Claude 내에서 Claude API를 호출하는 대화형 앱이 될 수 있습니다.
- 공유된 앱은 보는 사람이 자신의 Claude 계정으로 인증하며, 사용량은 각자의 구독에 반영되고 제작자는 비용을 부담하지 않으며 API 키가 필요 없습니다.
- Claude가 실제 코드를 생성하고 디버깅을 돕고, 누구나 artifact를 포크해 커스터마이즈할 수 있습니다.
- 당시 기준 제약으로 외부 API 호출 불가, 영구 저장소 없음, 텍스트 기반 completion API 한정 등이 명시됩니다.

## 번들 리소스
- Skill: Claude-powered artifacts의 동작 방식(공유/인증/과금)과 제약을 설명하는 재사용 가능한 제품 브리프
