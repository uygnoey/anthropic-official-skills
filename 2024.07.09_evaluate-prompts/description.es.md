# Evaluate prompts in the developer console

## ¿De qué trata este post?
Este post presenta la función Evaluate de la Anthropic Console para generar prompts, crear casos de prueba y comparar salidas entre versiones de prompts.

## ¿Cuándo es útil?
Es útil cuando necesitas un método repetible para probar cambios de prompt antes de ponerlos en producción, con ejecuciones por lotes y evaluación estructurada.

## Puntos clave
- La Console puede generar un prompt inicial a partir de una descripción de la tarea.
- Puedes importar casos de prueba desde CSV o generarlos automáticamente por variable de entrada.
- Puedes ejecutar toda la batería de pruebas y comparar resultados lado a lado entre versiones del prompt.
- Opcionalmente, expertos del dominio pueden calificar la calidad en una escala de 5 puntos para medir mejoras.
