# Evaluate prompts in the developer console

## What is this post?
This post introduces the Anthropic Console's Evaluate feature for generating prompts, creating test cases, and comparing outputs across prompt versions.

## When is it useful?
Useful when you want a repeatable way to test prompt changes before shipping, using batch test runs and structured evaluation.

## Key points
- Use the Console prompt generator to draft an initial prompt from a task description.
- Create a test suite by importing CSV test cases or generating them automatically per input variable.
- Run all test cases in one click, then compare outputs across prompt versions side by side.
- Optionally have subject matter experts (SMEs) score responses on a 5-point scale to track quality improvements.
