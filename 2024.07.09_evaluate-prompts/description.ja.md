# Evaluate prompts in the developer console

## この記事について
この記事は、Anthropic Console の Evaluate 機能を紹介し、プロンプト生成・テストケース生成・バージョン間の出力比較による改善の流れを説明します。

## どんなときに役立つか
本番投入前にプロンプトの変更を反復的に検証し、バッチ実行と評価で品質を確認したいときに役立ちます。

## 主なポイント
- Console のプロンプト生成機能で、タスク説明から初期プロンプトを作成できます。
- CSV 取り込みや入力変数ごとの要件に基づく自動生成でテストケースを用意できます。
- テストスイートをまとめて実行し、プロンプトの複数バージョンの出力を並べて比較できます。
- 必要に応じて SME が 5 段階で品質を採点し、改善度合いを把握できます。
