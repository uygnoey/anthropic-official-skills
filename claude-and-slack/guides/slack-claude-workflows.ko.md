# Slack에서 Claude 활용하기

## 1) Choose a mode
- Use **Claude in Slack** to get help directly in Slack channels, threads, or DMs.
- Use the **Slack connector in Claude** when you want Claude to search Slack messages and shared files for context.

## 2) Use Claude in Slack
- **Direct messages:** message **@Claude** for private help.
- **AI assistant panel:** open Claude from Slack's AI assistant header to get help without disrupting the channel.
- **Thread participation:** mention **@Claude** in a thread; Claude drafts a response privately so you can review before posting.

## 3) Connect Slack to Claude
- Ensure your workspace admin has enabled the connector for the organization.
- Set it up in the **Connectors** tab in Claude settings.

## 4) Common workflows
- Meeting preparation: pull recent updates and decisions.
- Project coordination: gather status updates and identify blockers.
- Onboarding support: summarize channel history and important context.
- Documentation: formalize thread outcomes into docs.
