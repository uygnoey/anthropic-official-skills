# What is this post?
This post describes two ways to use Claude with Slack: installing the Claude app in Slack for help in channels/threads, and connecting Slack to Claude so Claude can search conversations and files for relevant context.

# When is it useful?
It is useful when your team coordinates work in Slack and you want help drafting replies, summarizing discussions, preparing meetings, or turning thread decisions into documentation while staying in your workflow.

# Key points
- Two modes: Claude in Slack (app) and Slack connector in Claude.
- In threads, Claude drafts responses privately so you can review before posting.
- Claude only accesses channels and conversations you have permission to view.
- Suggested use cases include meeting prep, project coordination, onboarding support, and documentation.
