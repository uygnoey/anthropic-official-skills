# 이 글이 뭔가요
이 글은 Slack에서 Claude를 사용하는 두 가지 방법(Slack에 Claude 앱 설치로 채널/스레드에서 도움 받기, Claude 설정에서 Slack 커넥터를 연결해 대화/파일을 검색·참조하기)을 설명합니다.

# 언제 유용한가요
팀이 Slack에서 업무를 조율할 때, 답장 초안 작성, 대화 요약, 회의 준비, 스레드의 결정을 문서화하는 일을 워크플로우를 끊지 않고 처리하고 싶을 때 유용합니다.

# 핵심 포인트
- 두 가지 모드: Slack의 Claude(앱)와 Claude의 Slack 커넥터.
- 스레드에서는 Claude가 먼저 비공개로 초안을 작성해, 게시 전 검토할 수 있습니다.
- Claude는 사용자가 접근 권한이 있는 채널/대화만 접근합니다.
- 회의 준비, 프로젝트 조율, 온보딩 지원, 문서화 같은 활용 예시가 제시됩니다.
