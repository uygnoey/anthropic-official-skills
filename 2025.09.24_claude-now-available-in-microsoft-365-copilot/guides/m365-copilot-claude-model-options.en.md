## Overview
This guide summarizes where Claude shows up in Microsoft 365 Copilot according to the announcement.

## Where Claude is available
- Researcher: Microsoft’s reasoning agent can be powered by Claude Opus 4.1.
- Copilot Studio: Claude Sonnet 4 and Claude Opus 4.1 are available as model options for building enterprise-grade agents.

## Notes
- Availability is described as rolling out via Microsoft’s Frontier Program (opt-in).
- Anthropic notes the models are hosted outside Microsoft-managed environments and subject to Anthropic Terms and Conditions.
