## ¿De qué trata este post?
Anthropic resume el anuncio de Microsoft: **los modelos de Claude ya están disponibles en Microsoft 365 Copilot**, empezando por Researcher y Copilot Studio.

## ¿Cuándo es útil?
- Para usuarios empresariales que quieren acceder a Claude desde el entorno de productividad de Microsoft.
- Cuando una organización necesita elegir el modelo dentro de experiencias de Microsoft 365 Copilot (Researcher o Copilot Studio).

## Puntos clave
- Los clientes de Microsoft 365 Copilot pueden elegir **Claude Sonnet 4** y **Claude Opus 4.1** como opciones de modelo (en el despliegue, mediante el Frontier Program de Microsoft).
- El agente **Researcher** de Microsoft puede ser impulsado por **Claude Opus 4.1** para investigación de varios pasos.
- Los modelos de Claude también están disponibles en **Copilot Studio** para crear y gestionar agentes de nivel empresarial.
- Anthropic señala que los modelos se alojan fuera de los entornos gestionados por Microsoft y están sujetos a los Términos y Condiciones de Anthropic.
