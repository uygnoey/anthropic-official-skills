## What is this post?
This post announces Claude’s ability to create and edit files (spreadsheets, documents, slide decks, and PDFs) directly, producing ready-to-use outputs instead of only text.

## When is it useful?
Use it when you need a polished deliverable (e.g., an Excel model, a formatted report, or a slide deck) generated from instructions and/or uploaded source material.

## Key points
- Claude can generate and edit Excel spreadsheets, Word-style documents, PowerPoint slide decks, and PDFs.
- It can turn raw data into cleaned datasets, analysis, charts, and written insights.
- It supports cross-format work (e.g., turning a PDF report into slides).
- You can iteratively guide the work via chat, then download the finished file (and in some experiences, save to Google Drive).
