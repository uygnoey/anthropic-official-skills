## ¿De qué trata este post?
Este post anuncia que Claude ahora puede crear y editar archivos (hojas de cálculo, documentos, presentaciones y PDFs) directamente, devolviendo entregables listos para usar en lugar de solo texto.

## ¿Cuándo es útil?
Es útil cuando necesitas un entregable pulido (por ejemplo, un modelo en Excel, un informe con formato o una presentación) a partir de instrucciones y/o material fuente que subas.

## Puntos clave
- Claude puede generar y editar hojas de cálculo de Excel, documentos tipo Word, presentaciones de PowerPoint y PDFs.
- Puede convertir datos en bruto en datos limpios, análisis, gráficos y conclusiones por escrito.
- Permite trabajo entre formatos (por ejemplo, convertir un informe en PDF en diapositivas).
- Puedes guiar el proceso de forma iterativa por chat y luego descargar el archivo final (y, en algunas experiencias, guardarlo en Google Drive).
