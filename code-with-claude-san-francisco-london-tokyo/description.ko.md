## 이 글이 뭔가요
이 글은 Anthropic의 개발자 컨퍼런스인 Code with Claude가 샌프란시스코, 런던, 도쿄에서 열린다는 소식을 소개합니다.

## 언제 유용한가요
기술 세션, 라이브 데모, 워크숍, 오피스 아워를 통해 Claude Code의 최신 기능을 배우고 싶을 때(현장 참석 또는 라이브스트림 시청) 유용합니다.

## 핵심 포인트
- 도시 및 일정: 샌프란시스코(2026년 5월 6일), 런던(2026년 5월 19일), 도쿄(2026년 6월 10일).
- 현장 참석 신청이 열려 있으며, 참석자는 무작위 추첨으로 선정되어 4월 초에 안내됩니다.
- 세션은 라이브스트림 및 녹화본으로 누구나 시청할 수 있습니다.
