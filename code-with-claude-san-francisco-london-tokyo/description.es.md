## ¿De qué trata este post?
Este post anuncia Code with Claude, la conferencia para desarrolladores de Anthropic, que se amplía a San Francisco, Londres y Tokio.

## ¿Cuándo es útil?
Es útil si quieres aprender sobre Claude Code mediante sesiones técnicas, demos en vivo, talleres y horas de oficina, ya sea en persona (plazas limitadas) o por livestream.

## Puntos clave
- Ciudades y fechas: San Francisco (6 de mayo de 2026), Londres (19 de mayo de 2026), Tokio (10 de junio de 2026).
- Las solicitudes para asistir en persona están abiertas; los asistentes se eligen por sorteo y se notifica a principios de abril.
- Las sesiones se transmitirán en vivo y quedarán grabadas para que cualquiera pueda verlas.
