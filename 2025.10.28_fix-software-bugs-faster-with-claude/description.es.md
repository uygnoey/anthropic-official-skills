# Fix software bugs faster with Claude

## ¿De qué trata este post?
Este post propone un flujo práctico para depurar más rápido usando Claude.ai para análisis rápido y Claude Code para depuración autónoma en varios pasos dentro de un repositorio.

## ¿Cuándo es útil?
Es útil cuando necesitas pasar de síntomas (trazas, logs, tests inestables) a la causa raíz y a una corrección verificada, especialmente si el problema abarca muchos archivos o servicios.

## Puntos clave
- Empieza con Claude.ai para formular hipótesis a partir de trazas, logs y síntomas.
- Usa Claude Code para investigar en el repositorio, implementar correcciones con permiso y validar ejecutando tests.
- Reutiliza prompts claros para pedir causas probables ordenadas, tests de reproducción y planes de refactorización segura.

## Recursos incluidos
- Agent Skill: paquete de prompts de depuración y flujo paso a paso (con plantillas).
