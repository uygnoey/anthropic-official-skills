# Fix software bugs faster with Claude

## 이 글이 뭔가요
이 글은 Claude.ai로 빠르게 가설을 세우고, Claude Code로 코드베이스 전반을 자율적으로 조사·수정·테스트하는 방식으로 디버깅을 더 빠르게 하는 실전 워크플로를 설명합니다.

## 언제 유용한가요
스택 트레이스/로그/불안정한 테스트 같은 증상에서 출발해 원인을 좁히고, 테스트로 검증된 수정까지 빠르게 도달해야 할 때(특히 변경 범위가 넓을 때) 유용합니다.

## 핵심 포인트
- Claude.ai로 증상 자료를 넣어 가능한 원인을 가설 형태로 정리합니다.
- Claude Code로 저장소 전체를 조사하고, 권한 확인 후 수정안을 적용한 뒤 테스트로 검증합니다.
- ‘가능한 원인을 가능성 순으로 정렬’, ‘재현 테스트 작성’, ‘안전한 리팩터링’ 같은 재사용 프롬프트를 활용합니다.

## 번들 리소스
- Agent Skill: 디버깅 프롬프트 팩 + 단계별 워크플로(템플릿 포함)
