# Fix software bugs faster with Claude

## この記事について
この記事は、Claude.ai で素早く分析し、Claude Code でコードベース全体を対象に自律的な複数ステップのデバッグを行うことで、バグ修正を高速化する実践ワークフローを紹介します。

## どんなときに役立つか
スタックトレース、ログ、フレーキーテストなどの症状から原因を絞り込み、検証済みの修正に到達したいとき（特に多数のファイルやサービスにまたがる場合）に役立ちます。

## 主なポイント
- まず Claude.ai で症状情報から仮説を立てます。
- Claude Code でリポジトリを横断して調査し、許可を得て修正を適用し、テスト実行で検証します。
- 可能性順の原因リスト、再現テスト作成、安全なリファクタリング計画など、再利用できるプロンプトを活用します。
