## 이 글이 뭔가요
Anthropic이 **Claude for Enterprise**(엔터프라이즈 플랜)를 발표하며, 보안과 조직 단위 협업을 중심으로 한 기능(500K 컨텍스트, 더 큰 사용량, 관리/보안 기능 등)을 소개합니다.

## 언제 유용한가요
- 조직에서 Claude를 도입할 때 **SSO, 역할 기반 권한, 감사 가능성** 같은 중앙 관리가 필요할 때
- 긴 문서/다수의 트랜스크립트/상당한 규모의 코드베이스 등 **대규모 내부 자료를 한 번에 다뤄야** 할 때

## 핵심 포인트
- 엔터프라이즈 플랜은 **500K 컨텍스트**, 더 높은 사용량, 관리 기능을 포함합니다.
- **SSO/도메인 캡처**, **역할 기반 접근 제어**를 제공하며, **감사 로그**와 **SCIM**은 추후 제공 예정으로 언급됩니다.
- (발표 시점 기준) GitHub 리포지토리를 Claude와 동기화하는 **네이티브 GitHub 연동(베타)**을 언급합니다.
- Anthropic은 고객의 대화 및 콘텐츠로 Claude를 학습시키지 않는다고 밝힙니다.
