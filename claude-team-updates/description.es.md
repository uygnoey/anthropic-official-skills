## ¿De qué trata este post?
Este post anuncia cambios en el plan Claude Team, incluyendo nuevos precios y un descuento anual para los asientos premium.

## ¿Cuándo es útil?
Es útil si estás evaluando Claude Team para un equipo en crecimiento y quieres entender la estructura de precios actualizada y lo que incluye el plan.

## Puntos clave
- Asientos Standard: 20 USD/mes con facturación anual (o 25 USD/mes con facturación mensual).
- Asientos Premium: 100 USD/mes con facturación anual (o 125 USD/mes con facturación mensual).
- Team incluye un espacio de trabajo compartido, controles de administración, gestión de facturación, privacidad por defecto, conectores para acceder al conocimiento interno y Claude Code incluido en cada asiento.
