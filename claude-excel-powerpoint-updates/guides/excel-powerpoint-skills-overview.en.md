## Overview
This guide summarizes what changed in Claude for Excel and Claude for PowerPoint, focusing on skills, cross-file context, and persistent instructions.

## What’s new
- Shared conversation context across all open Excel and PowerPoint files.
- Skills available inside the add-ins, turning multi-step workflows into a one-click action.
- Persistent “Instructions” for preferences that should always apply.

## Starter workflows mentioned in the post
- Excel: model audit; LBO/DCF/3-statement templates; comparable company analysis; cleaning messy data.
- PowerPoint: competitive landscape decks; updating an existing deck; reviewing deck consistency and polish.
