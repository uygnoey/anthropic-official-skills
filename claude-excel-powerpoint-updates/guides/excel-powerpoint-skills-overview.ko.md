## 개요
이 가이드는 Claude for Excel 및 Claude for PowerPoint의 변경점을 ‘스킬’, 파일 간 컨텍스트 공유, 그리고 영구 설정(Instructions) 중심으로 정리합니다.

## 무엇이 달라졌나요
- 열려 있는 Excel/PowerPoint 파일 전반에서 대화 컨텍스트를 공유합니다.
- 애드인 내부에서 스킬을 실행해 여러 단계 워크플로를 한 번에 수행할 수 있습니다.
- 항상 적용되어야 하는 선호 설정을 ‘Instructions’로 영구화할 수 있습니다.

## 글에서 언급된 스타터 워크플로
- Excel: 모델 감사; LBO/DCF/3-스테이트먼트 템플릿; 컴프스 분석; 데이터 정리.
- PowerPoint: 경쟁 구도 덱 제작; 기존 덱 업데이트; 일관성/문장 다듬기 점검.
