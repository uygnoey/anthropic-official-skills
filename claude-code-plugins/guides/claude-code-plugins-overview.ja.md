## Claude Code プラグインとは？
プラグインは、Claude Code のカスタマイズをまとめて配布・共有するための軽量なバンドルです。

## プラグインに含められるもの
プラグインには次の拡張ポイントを任意に組み合わせて同梱できます。
- スラッシュコマンド
- サブエージェント
- MCP サーバー
- フック

## なぜプラグインを使うのか
- チームのベストプラクティスを環境として標準化する。
- デバッグ／デプロイ／テスト用ハーネスなどのワークフローを共有する。
- MCP サーバー経由でツールを接続する。
- 必要なときだけ有効化して、システムプロンプトのコンテキストを抑える。

## はじめ方
Claude Code から（パブリックベータ）:
1. マーケットプレイス追加: `/plugin marketplace add <user-or-org>/<repo-name>`
2. プラグイン導入: `/plugin install <plugin-name>`
