## What is a Claude Code plugin?
A plugin is a lightweight bundle used to package and share Claude Code customizations.

## What can a plugin include?
A plugin can bundle any combination of:
- slash commands
- subagents
- MCP servers
- hooks

## Why use plugins?
- Standardize environments (team best practices).
- Share workflows (debugging, deployments, testing harnesses).
- Connect tools (via MCP servers).
- Toggle capabilities on/off to reduce system prompt context.

## Getting started
From inside Claude Code (public beta):
1. Add a marketplace: `/plugin marketplace add <user-or-org>/<repo-name>`
2. Install a plugin: `/plugin install <plugin-name>`
