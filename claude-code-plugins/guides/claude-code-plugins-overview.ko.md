## Claude Code 플러그인이란?
플러그인은 Claude Code의 커스터마이징을 묶어 배포·공유하기 위한 가벼운 번들입니다.

## 플러그인에 무엇을 넣을 수 있나요?
플러그인은 다음 확장 포인트를 조합해 포함할 수 있습니다.
- 슬래시 커맨드
- 서브에이전트
- MCP 서버
- 훅

## 왜 플러그인을 쓰나요?
- 팀 베스트 프랙티스를 기반으로 환경을 표준화합니다.
- 디버깅/배포/테스트 하네스 같은 워크플로를 공유합니다.
- MCP 서버로 도구를 연동합니다.
- 필요할 때만 켜서 시스템 프롬프트 컨텍스트를 줄입니다.

## 시작하기
Claude Code 안에서(퍼블릭 베타):
1. 마켓플레이스 추가: `/plugin marketplace add <user-or-org>/<repo-name>`
2. 플러그인 설치: `/plugin install <plugin-name>`
