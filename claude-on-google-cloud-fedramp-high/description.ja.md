# Claude on Google Cloud’s Vertex AI: FedRAMP High and IL2 Authorized

## この記事について
本記事は、Google Cloud Vertex AI 上で Claude モデルが FedRAMP High と DoD Impact Level 2（IL2）のワークロード向けに利用可能（認可）になったことを発表し、厳格なコンプライアンス要件の下でも連邦・国防関連組織が Claude を活用できることを説明します。

## どんなときに役立つか
Google Cloud の規制環境（FedRAMP High/IL2）で Claude を利用できるか評価・説明する必要があるときや、Vertex AI 上での社内導入計画を立てるときに役立ちます。

## 主なポイント
- Google Cloud Vertex AI にて、FedRAMP High および IL2 まで認可された環境で Claude モデルを利用できます。
- 政府向けのコンプライアンス制御として Assured Workloads が言及されています。
- インフラ運用を不要にしつつセキュリティ要件を満たすため、管理型／サーバレスの API アクセスを強調しています。
- FedRAMP High と IL2 対応は、将来的なより高いレベル（例：IL5）への足がかりとして位置付けられています。
