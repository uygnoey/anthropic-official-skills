# Claude on Google Cloud’s Vertex AI: FedRAMP High and IL2 Authorized

## 이 글이 뭔가요
이 글은 Google Cloud Vertex AI에서 Claude 모델이 FedRAMP High 및 DoD Impact Level 2(IL2) 워크로드에 대해 사용 가능(인가)해졌음을 발표하며, 엄격한 규정 준수 요구사항을 충족해야 하는 연방·국방 조직이 Claude를 활용할 수 있음을 설명합니다.

## 언제 유용한가요
Google Cloud의 규제 환경(FedRAMP High/IL2)에서 Claude 사용 가능 여부를 평가·커뮤니케이션해야 하거나, Vertex AI에서 내부 도입을 계획할 때 유용합니다.

## 핵심 포인트
- Google Cloud Vertex AI에서 FedRAMP High 및 IL2 수준까지 인가된 환경에서 Claude 모델을 사용할 수 있습니다.
- 정부 워크로드용 규정 준수 제어로 Assured Workloads가 언급됩니다.
- 보안 요구사항을 충족하면서 인프라를 직접 운영하지 않도록 관리형/서버리스 API 접근을 강조합니다.
- FedRAMP High 및 IL2 지원을 향후 더 높은 수준(예: IL5) 인가로 가는 진전으로 위치づけ합니다.

## 번들 리소스
- Skill: (원문에 구체 CLI 명령은 없으므로) 내부 도입 검토에 재사용 가능한 간단 체크리스트
