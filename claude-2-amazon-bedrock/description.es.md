## ¿De qué trata este post?
Este post anuncia que Claude 2 está disponible en Amazon Bedrock, el servicio gestionado de AWS para acceder a modelos fundacionales mediante una API.

## ¿Cuándo es útil?
Es útil si estás evaluando opciones para desplegar Claude en entornos de AWS, o si buscas ejemplos empresariales con requisitos de seguridad y cumplimiento.

## Puntos clave
- Claude 2 ya está disponible en Amazon Bedrock.
- El post destaca casos de uso empresariales (LexisNexis, Lonely Planet, Ricoh USA) y enfatiza la seguridad y la escalabilidad.
- No incluye una guía paso a paso ni ejemplos concretos de configuración.
