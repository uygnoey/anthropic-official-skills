## ¿De qué trata este post?
Este post anuncia el soporte de servidores MCP remotos en Claude Code, permitiendo conexiones seguras a herramientas y fuentes de datos externas sin ejecutar servidores locales.

## ¿Cuándo es útil?
Es útil cuando quieres que Claude Code acceda a herramientas de terceros (y a sus recursos) con una configuración de bajo mantenimiento, evitando gestionar servidores locales.

## Puntos clave
- Claude Code puede conectarse a servidores MCP remotos para acceder tanto a herramientas como a recursos expuestos por el servidor.
- Los servidores MCP remotos reducen el mantenimiento porque los proveedores gestionan actualizaciones, escalado y disponibilidad.
- Claude Code incluye soporte OAuth nativo para servidores MCP remotos, reduciendo la necesidad de claves API o credenciales almacenadas.
