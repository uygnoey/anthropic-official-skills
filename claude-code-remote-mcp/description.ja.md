## この記事について
この記事は、Claude Code がリモート MCP サーバーをサポートしたことを告知します。ローカルサーバーを運用せずに、外部ツールやデータソースへ安全に接続できます。

## どんなときに役立つか
Claude Code にサードパーティのツール（およびリソース）コンテキストを取り込ませつつ、ローカルサーバー管理を避けて低メンテナンスで運用したいときに役立ちます。

## 主なポイント
- Claude Code はリモート MCP サーバーが公開するツールとリソースの両方にアクセスできます。
- ベンダーが更新・スケーリング・可用性を管理するため、運用負荷を下げられます。
- リモート MCP 用のネイティブ OAuth により、API キー管理や資格情報の保存を減らせます。
