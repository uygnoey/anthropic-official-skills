# Reusable prompt snippets

These snippets are taken verbatim or lightly structured from prompts mentioned in the post.

## Bug-fix request
"Can you fix this bug? This is the behavior I'm seeing"

## Onboarding / codebase understanding
"Read this repository (including CLAUDE.md files). Identify which files are most relevant to my task and explain the key dependencies and data flows."

## Incident troubleshooting
"Given this stack trace and the related documentation, trace the control flow and identify likely causes. Propose mitigations and the next debugging steps."

## Iterative development loop
"Implement this change, run the tests, and iterate until the test suite passes. Explain what you changed at each iteration."
