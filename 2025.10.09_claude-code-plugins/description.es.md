## ¿De qué trata este post?
Este post presenta los plugins de Claude Code: una forma ligera de empaquetar y compartir personalizaciones de Claude Code que se pueden instalar y activar/desactivar desde Claude Code.

## ¿Cuándo es útil?
Es útil cuando quieres estandarizar o compartir una configuración de Claude Code (por ejemplo, flujos de trabajo de equipo o utilidades por proyecto) sin copiar configuraciones manualmente entre entornos.

## Puntos clave
- Los plugins pueden incluir cualquier combinación de puntos de extensión: comandos con barra, subagentes, servidores MCP y hooks.
- Están diseñados para activarse solo cuando se necesitan, reduciendo el contexto y la complejidad del prompt.
- Se pueden distribuir mediante marketplaces.
- Se instalan desde Claude Code con el comando `/plugin` (beta pública).
