## 이 글이 뭔가요
이 글은 Claude Code 플러그인을 소개합니다. 플러그인은 Claude Code의 커스터마이징(확장 포인트)을 한 번에 설치·공유할 수 있도록 묶어 배포하는 가벼운 번들 형식이며, Claude Code 안에서 켜고 끌 수 있습니다.

## 언제 유용한가요
팀이나 프로젝트에서 동일한 Claude Code 작업 환경(예: 테스트 하네스, 코드 리뷰 훅, 도구 연동)을 표준화하거나 공유하려고 할 때 유용합니다. 각자 설정을 수동으로 복사하지 않아도 됩니다.

## 핵심 포인트
- 플러그인은 슬래시 커맨드, 서브에이전트, MCP 서버, 훅 등 확장 포인트를 조합해 번들링할 수 있습니다.
- 필요할 때만 플러그인을 활성화해 시스템 프롬프트 컨텍스트와 복잡도를 줄이도록 설계되었습니다.
- 마켓플레이스를 통해 배포·발견할 수 있습니다.
- Claude Code에서 `/plugin` 명령으로 설치할 수 있습니다(퍼블릭 베타).
