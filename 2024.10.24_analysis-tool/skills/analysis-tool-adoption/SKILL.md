## Instructions
- Enable the analysis tool from Claude.ai feature previews.
- Use the analysis tool when you need mathematically precise calculations or to analyze data with exact results.

## Examples
- “Enable the analysis tool and compute summary statistics for this dataset.”
- “Use the analysis tool to calculate the exact percentage change between these two values.”
