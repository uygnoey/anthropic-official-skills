## この記事について
    この記事は「Advancing Claude for Excel and PowerPoint」の更新内容を告知し、新機能と提供状況をまとめています。

    ## どんなときに役立つか
    Excel/PowerPoint アドインで可能なこと、再利用可能なワークフロー（skills）と永続的な設定（Instructions）が日々の作業にどう効くか把握したいときに役立ちます。

    ## 主なポイント
    - Excel and PowerPoint add-ins share full conversational context across open files.
- Skills are available inside the add-ins and can turn entire workflows into a one-click action.
- Instructions let you set persistent app-level preferences (formatting, slide rules, hardcoded assumption flags).
- Starter skills exist for common finance and presentation workflows; org-wide skills work out of the box.
- Add-ins can run via Claude account or through Amazon Bedrock, Google Cloud Vertex AI, or Microsoft Foundry.

    ## 同梱リソース
    - Skill bundle: a starter set of Excel workflows (model audit; LBO/DCF/3-statement templates; comps; data cleaning).
- Skill bundle: a starter set of PowerPoint workflows (competitive landscape; deck updates; deck consistency review).
- Concept: persistent “Instructions” that apply automatically to add-in usage.
- Mention: Financial Analysis plugin distributes starter skills to both add-ins.
