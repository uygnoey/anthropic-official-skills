## What is this post?
This post shares how three Y Combinator startups use Claude Code to ship faster, manage context deliberately, and scale agentic development workflows.

## When is it useful?
Use it when you want a repeatable way to break work into research/planning/implementation sessions, and when you’re considering sub-agent workflows for parallel investigation.

## Key points
- Startups highlight a three-phase workflow: research, planning, implementation (often in separate sessions).
- Context management matters: start new sessions when context becomes contradictory or noisy.
- Sub-agent workflows can parallelize codebase research and handle different data types.
