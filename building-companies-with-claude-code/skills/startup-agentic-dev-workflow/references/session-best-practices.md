# Session best practices for Claude Code (from startup examples)

## Core practices
- Separate research, planning, and implementation into discrete sessions.
- Be deliberate about context management, and start a new session when context becomes contradictory or noisy.
- Monitor early tool calls and interrupt unhelpful behavior quickly.

## Notes from the source post
The post mentions:
- Using different model choices for different phases (research vs implementation).
- Using subagents for parallel investigation.
