## この記事について
この記事は、Y Combinatorの3社がClaude Codeを使って開発を高速化し、コンテキストを意図的に管理し、エージェント的な開発ワークフローをスケールさせている方法を紹介します。

## どんなときに役立つか
リサーチ／計画／実装をセッション単位で分ける再現可能な進め方が欲しいとき、また並列調査のためのサブエージェント・ワークフローを検討しているときに役立ちます。

## 主なポイント
- 3フェーズ（リサーチ→計画→実装）の進め方を紹介し、しばしば各フェーズを別セッションで行います。
- コンテキスト管理が重要で、矛盾やノイズが増えたら新しいセッションを開始します。
- サブエージェントによりコードベース調査を並列化し、データ種別ごとに作業を分担できます。
