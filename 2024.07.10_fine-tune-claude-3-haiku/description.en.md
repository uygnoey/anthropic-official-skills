# Fine-tune Claude 3 Haiku in Amazon Bedrock

## What is this post?
This post announces and explains fine-tuning for Claude 3 Haiku in Amazon Bedrock, including what it enables and how to get access.

## When is it useful?
Use it when you need a customized Claude 3 Haiku that performs better on specialized tasks (classification, domain-specific interpretation, structured outputs) while keeping latency and cost low.

## Key points
- Fine-tuning uses high-quality prompt–completion pairs to specialize Claude 3 Haiku.
- Fine-tuning runs within your AWS environment; your training data stays in your AWS account.
- You can test, iterate, and deploy via the Amazon Bedrock console or API.
