# Fine-tune Claude 3 Haiku in Amazon Bedrock

## ¿De qué trata este post?
Este post anuncia y explica el ajuste fino (fine-tuning) de Claude 3 Haiku en Amazon Bedrock: qué permite y cómo solicitar acceso.

## ¿Cuándo es útil?
Es útil cuando necesitas un Claude 3 Haiku personalizado que rinda mejor en tareas especializadas (clasificación, interpretación por dominio, salidas estructuradas) manteniendo baja la latencia y el coste.

## Puntos clave
- El ajuste fino utiliza pares de prompt–completion de alta calidad para especializar Claude 3 Haiku.
- El ajuste fino ocurre dentro de tu entorno de AWS; tus datos de entrenamiento permanecen en tu cuenta.
- Puedes probar, iterar y desplegar con la consola o la API de Amazon Bedrock.

## Recursos incluidos
- Agent Skill: lista de verificación para preparar datos prompt–completion y evaluar el comportamiento tras el ajuste fino.
