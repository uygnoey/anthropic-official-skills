# Fine-tune Claude 3 Haiku in Amazon Bedrock

## この記事について
この記事は、Amazon Bedrock で Claude 3 Haiku をファインチューニングできるようになったことを紹介し、何ができるのかと利用開始（アクセス申請）方法を説明します。

## どんなときに役立つか
分類、ドメイン特化の解釈、構造化出力などの特定タスクに最適化しつつ、レイテンシとコストを低く保ちたい場合に役立ちます。

## 主なポイント
- 高品質な prompt–completion ペアで Claude 3 Haiku を特化させます。
- ファインチューニングは AWS 環境内で行われ、学習データは顧客の AWS アカウント内に留まります。
- Amazon Bedrock のコンソールまたは API でテスト、反復、デプロイを行えます。
