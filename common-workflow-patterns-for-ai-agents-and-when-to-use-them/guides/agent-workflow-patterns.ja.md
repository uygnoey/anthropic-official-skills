# エージェントのワークフローパターン（逐次、並列、評価者–最適化）

## パターンごとの使いどころ
- **逐次（Sequential）**: 前段の出力に依存する処理があるとき。
- **並列（Parallel）**: サブタスクが独立で、遅延を減らしたいとき。
- **評価者–最適化（Evaluator–optimizer）**: 初回の品質が不十分で、評価基準を定義できるとき。
