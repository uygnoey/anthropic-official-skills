# 에이전트 워크플로 패턴(순차, 병렬, 평가자–최적화)

## 패턴별 사용 시점
- **순차(Sequential)**: 단계 간 의존성이 있을 때.
- **병렬(Parallel)**: 서브태스크가 독립적이고 지연 시간이 중요할 때.
- **평가자–최적화(Evaluator–optimizer)**: 1차 결과 품질이 부족하고 평가 기준을 정의할 수 있을 때.
