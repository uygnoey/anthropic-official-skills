## ¿De qué trata este post?
Este post presenta una guía descargable para crear skills de Claude, con planificación, estructura, pruebas/iteración y distribución.

## ¿Cuándo es útil?
Es útil cuando quieres que Claude aplique tus flujos de trabajo de forma consistente empaquetando instrucciones, scripts y referencias en una skill reutilizable.

## Puntos clave
- Las skills son carpetas empaquetadas que enseñan un flujo de trabajo repetible.
- Buenas skills usan divulgación progresiva: mantener SKILL.md conciso y enlazar a referencias/scripts incluidos.
- La guía enfatiza planificar casos de uso concretos, probar activación/falsos positivos e iterar a partir de fallos reales.

## Recursos incluidos
- Un Agent Skill que resume la estructura recomendada de SKILL.md y una lista de verificación de pruebas.
