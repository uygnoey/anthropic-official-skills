## この記事について
この記事は、Claude がアプリ、ブラウザ、開発ツールを含む「あなたのコンピューター」を操作してタスクを完了できるようになったことと、スマホからタスクを割り当てる Dispatch を紹介します。

## どんなときに役立つか
コネクタのような精密ツールが使えない／足りないため UI 操作が必要な場面で、既存のアプリやワークフロー内の複数ステップ作業を Claude に任せたいときに役立ちます。

## 主なポイント
- Claude Cowork と Claude Code は画面上でクリックやスクロールを行い、アプリ内の作業を進められます。
- 可能なら Slack や Google Calendar などのコネクタを優先し、必要な場合にのみ画面・マウス・キーボード制御に切り替えます。
- 実行前に明示的な許可を求め、プロンプトインジェクション検知や一部アプリの既定制限などの安全策があります。
- Dispatch により、スマホでタスクを割り当て、デスクトップで同じ会話を継続できます。
