# How enterprises are building AI agents in 2026

## What is this post?
This post summarizes findings from a survey of 500+ technical leaders about how enterprises are deploying AI agents, what use cases are common, what challenges persist, and where organizations expect to expand next.

## When is it useful?
- When you need an executive-level snapshot of agent adoption, common workflows, and reported ROI.
- When planning agent programs and want to anticipate integration, data, and change-management obstacles.

## Key points
- Agents are already used for multi-stage workflows and are expected to expand toward more complex, cross-functional use cases.
- Coding assistance is widely adopted, and many organizations use AI in production code.
- Common enterprise agent applications include data analysis/reporting and internal process automation.
- The post highlights recurring blockers: integration with existing systems, data access/quality, and change management.
