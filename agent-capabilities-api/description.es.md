## ¿De qué trata este post?
Este post presenta nuevas capacidades de la API para crear mejores agentes de IA: una herramienta de ejecución de código, un conector MCP, una API de Archivos y caché de prompts extendida.

## ¿Cuándo es útil?
Es útil cuando necesitas agentes que analicen datos con Python, se conecten a sistemas externos mediante servidores MCP, almacenen y reutilicen archivos entre sesiones o reduzcan latencia y costos en prompts largos gracias al almacenamiento en caché.

## Puntos clave
- La ejecución de código permite que Claude ejecute Python en un entorno aislado para análisis e iteración.
- El conector MCP permite conectar Claude a servidores MCP remotos y usar sus herramientas sin escribir código de integración del cliente.
- La API de Archivos permite subir documentos una vez y reutilizarlos entre sesiones.
- La caché de prompts extendida ofrece opciones de TTL más largas (incluido hasta 60 minutos) para reducir costos y mejorar latencia en prompts largos repetidos.
