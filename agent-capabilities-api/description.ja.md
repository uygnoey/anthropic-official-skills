## この記事について
この記事では、より良いAIエージェントを構築するための新しいAPI機能（コード実行ツール、MCPコネクタ、Files API、拡張プロンプトキャッシュ）を紹介します。

## どんなときに役立つか
Pythonでデータ分析をしたい、MCPサーバー経由で外部システムに接続したい、ファイルをセッション間で保存・再利用したい、長いプロンプトをキャッシュして遅延やコストを下げたい、といった場合に役立ちます。

## 主なポイント
- コード実行により、Claudeがサンドボックス環境でPythonを実行して分析・反復ができます。
- MCPコネクタにより、クライアント側の統合作業なしでリモートMCPサーバーに接続し、そのツールを利用できます。
- Files APIにより、文書を一度アップロードして複数セッションで再利用できます。
- 拡張プロンプトキャッシュは、より長いTTL（最大60分を含む）を提供し、繰り返し使う長文プロンプトのコストと遅延を削減します。
