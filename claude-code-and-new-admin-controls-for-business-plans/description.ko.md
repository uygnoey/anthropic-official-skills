## 이 글이 뭔가요
이 글은 Team·Enterprise 고객이 프리미엄 시트를 통해 Claude Code를 사용할 수 있게 되었음을 알리고, 새로운 관리자 제어(관리 정책), 사용량 분석, Compliance API를 소개합니다.

## 언제 유용한가요
조직에서 Claude와 Claude Code를 운영·관리하며 시트/권한 관리, 비용 한도, 정책 강제, 분석 지표, 감사·거버넌스 연동이 필요할 때 유용합니다.

## 핵심 포인트
- Team·Enterprise 플랜의 프리미엄 시트에 Claude Code가 포함됩니다.
- 관리자는 시트 할당을 관리하고, 조직/개인 단위 지출 한도를 설정하며, 사용자 전반에 정책 설정을 강제할 수 있습니다.
- 사용량 분석에 Claude Code 관련 지표(예: 수락된 코드 라인 수, 제안 수락률)가 포함됩니다.
- Compliance API는 사용량 데이터와 고객 콘텐츠에 대한 프로그램적 접근을 제공해 감사·거버넌스를 지원합니다.
- 이 글에는 구체적인 API 요청 예시나 JSON 설정 스니펫이 포함되어 있지 않습니다.
