## この記事について
この記事は、Team／Enterprise の顧客がプレミアムシートを通じて Claude Code を利用できるようになったことを発表し、新しい管理者向けコントロール、利用状況アナリティクス、Compliance API を紹介します。

## どんなときに役立つか
組織で Claude と Claude Code を管理しており、シート管理、支出上限、ポリシー適用、分析指標、ガバナンス／監査連携が必要な場合に役立ちます。

## 主なポイント
- Team／Enterprise プランのプレミアムシートに Claude Code が含まれます。
- 管理者はシート割り当ての管理、組織／個人単位の支出上限設定、全ユーザーへのポリシー設定の適用が可能です。
- 利用状況アナリティクスに Claude Code 固有の指標（例：受け入れられたコード行数、提案の受け入れ率）が追加されます。
- Compliance API は、監査とガバナンスのために、利用状況データや顧客コンテンツへのプログラム的アクセスを提供します。
- この記事には、具体的な API リクエスト例や JSON 設定スニペットは含まれていません。
