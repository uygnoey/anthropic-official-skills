## ¿De qué trata este post?
Este post anuncia que los clientes de Team y Enterprise pueden acceder a Claude Code mediante asientos premium, y presenta nuevos controles administrativos, analíticas de uso y una Compliance API.

## ¿Cuándo es útil?
Es útil si administras Claude y Claude Code en una organización y necesitas gestión de asientos, controles de gasto, políticas, analíticas o integraciones para gobernanza y auditoría.

## Puntos clave
- Los asientos premium de los planes Team y Enterprise incluyen Claude Code.
- Los administradores pueden gestionar la asignación de asientos, establecer límites de gasto y aplicar políticas a todos los usuarios.
- Las analíticas de uso incluyen métricas específicas de Claude Code (por ejemplo, líneas de código aceptadas y tasa de aceptación de sugerencias).
- La Compliance API ofrece acceso programático a datos de uso y contenido del cliente para apoyar auditoría y gobernanza.
- El post no incluye ejemplos concretos de solicitudes a la API ni fragmentos de configuración en JSON.
