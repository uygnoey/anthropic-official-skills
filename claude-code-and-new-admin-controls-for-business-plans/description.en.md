## What is this post?
This post announces that Team and Enterprise customers can access Claude Code via premium seats, and introduces new administrative controls, usage analytics, and a Compliance API.

## When is it useful?
It’s useful if you administer Claude and Claude Code in an organization and need seat management, spend controls, policy enforcement, analytics, or governance/audit integrations.

## Key points
- Premium seats for Team and Enterprise plans include Claude Code.
- Admins can manage seat allocation, set spend limits, and enforce policy settings across users.
- Usage analytics add Claude Code-specific metrics (e.g., accepted lines of code, suggestion acceptance rate).
- The Compliance API provides programmatic access to usage data and customer content to support auditing and governance.
- The post does not include concrete API request examples or JSON configuration snippets.
